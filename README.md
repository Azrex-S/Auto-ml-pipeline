# Auto-ml-pipeline
An AutoML pipeline automates data processing, model selection, and tuning to quickly build and evaluate machine learning models with minimal manual effort.
This repository contains a work in progress AutoML pipeline. The goal is to automate the machine learning workflow handling data preprocessing, model selection, and tuning to quickly build and test models.

Note: This project is still under development. Features, code structure, and examples may change as I continue working on it.
